ADGI Week 2025

A visual system built for momentum, dialogue, and creative exchange
Modular, scalable, and built to move across formats

Overview

Event Identity | Visual System | Digital + Print Rollout

ADGI Week 2025 is a design week identity concept built to feel alive and adaptable. The system is designed around strong typographic hierarchy, flexible layout rules, and a visual rhythm that can scale from posters to social templates to venue signage without losing clarity.

Design Intent

A system that stays consistent while the content changes daily. The grid stays structured, the type stays loud and readable, and the layout can flex between minimal announcements and dense schedules without turning messy.

The goal wasn’t to make something decorative. It was to build a system that can actually survive a real event rollout. Everything is modular on purpose: consistent rules, flexible layouts, and enough variation to keep the energy up without losing the identity.

If this project feels like it’s moving even when it’s still, that’s the point.
